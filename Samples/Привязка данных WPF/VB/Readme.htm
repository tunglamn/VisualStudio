Примечание. Этот пример предназначен для Microsoft .NET Framework 3.5.

Это пример Windows Presentation Framework, использующий модель EDM (Entity Data Model) для подключения к простой системе управления заказами в базе данных SQL Server. В этом примере имеется несколько форм WPF, демонстрирующих различные возможности привязки данных. Пример включает следующие компоненты:

·         Учебная база данных SQL Server: OMS. Скрипт SQL, включенный в этот пример, может использоваться для создания и заполнения базы данных.

·         Уровень доступа к данным. Этот проект библиотеки классов использует модель EDM (Entity Data Model) для сопоставления различных таблиц в базе данных. Он также предоставляет некоторые алгоритмы проверки.

·         Проект приложения WPF. Использует уровень доступа к данных и отображает их. Проект содержит несколько форм WPF, предназначенных для демонстрации различных возможностей работы с данными, например:

o    Простой список элементов

o    DataGrid

o    Представление «Подробности» с кнопками перехода

o    Основное/Подробности

o    Таблицы подстановки

o    Форматирование и проверка

 

Запуск примера

1.     Откройте проект примера.

2.     Под узлом «Решение» разверните папку Элементы решения и откройте CreateDatabaseOMS.sql.

3.     В редакторе TSQL щелкните правую кнопку мыши и в контекстном меню выберите команду Выполнить SQL.

4.     При появлении запроса выберите режим соединения, а затем выберите существующее или создайте новое подключение к серверу SQL Server.

5.     Закройте редактор TSQL

6.     Нажмите клавишу F5, чтобы выполнить построение и запустить пример

Требования

·         Microsoft SQL Server 2005 или более поздняя версия
